# Live Printer Feeds Dashboard

Real-time dashboard for monitoring FabLab 3D printer webcam feeds.

## Features

- 🖼️ **Live Camera Feeds** - Real-time views of Core 1 printers (XLs to be added)
- 🔍 **Click to Enlarge** - Click any feed to view full-screen with instant refresh
- ⚙️ **Customizable Settings**
  - Toggle header shimmer animation on/off
  - Adjust refresh interval (1-60 seconds)
  - Settings persist across sessions

## Usage

### Direct File
Simply open `fablab_live_printer_view.html` in any web browser.


## Requirements

- Access to uni network or VPN (camera feeds are network-restricted)
- Modern web browser (Chrome, Firefox, Safari, Edge)

## Keyboard Shortcuts

- `Escape` - Close enlarged view
